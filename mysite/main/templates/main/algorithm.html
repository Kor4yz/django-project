Раздел 1 — Условие задачи

Митя хочет приобрести картины, однако у него есть купюры только номиналом 500 у.е.

Напишите программу, которая поможет определить, какие картины он может купить без получения сдачи (то есть цена картины должна быть кратна 500).

Формат ввода:

  • На одной строке через запятую и пробел — данные о картинах: Название - цена
  • Минимум одна картина

Формат вывода:

  • Названия картин, которые можно купить без сдачи
  • В алфавитном порядке, через запятую и пробел
⬆ К началу страницы

Раздел 2 — Код решения (Python)

Ниже представлено решение на Python, встроенное в Django-приложение:

def solve_paintings(paintings, bill=500): """ Определяет картины, которые можно купить купюрами по 500 у.е. без сдачи. :param paintings: список словарей {'name': str, 'price': int} :param bill: номинал купюры (по умолчанию 500) :return: отсортированный список названий картин """ result = sorted( [p['name'] for p in paintings if p['price'] % bill == 0], key=lambda x: x.lower() ) return result # Пример вызова: # paintings = [ # {'name': 'Чёрный квадрат', 'price': 23000}, # {'name': 'Ночь', 'price': 17000}, # {'name': 'Девятый вал', 'price': 25000}, # ] # solve_paintings(paintings) # → ['Девятый вал', 'Ночь', 'Чёрный квадрат']

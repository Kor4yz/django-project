# Учебный Django-проект

## Запуск проекта

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Запустить сервер разработки
```bash
python manage.py runserver
```

### 3. Открыть в браузере
```
http://127.0.0.1:8000/
```

## Структура страниц

| URL | Описание |
|-----|----------|
| `/` | Главная страница |
| `/about/` | О себе + сокурсники |
| `/education/` | Образовательная программа |
| `/algorithm/` | Алгоритмическая задача (картины) |
